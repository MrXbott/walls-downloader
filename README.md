## walls-downloader
Это маленькая CLI утилита для скачивания обоев для рабочего стола с сайта [SmashingMagazine](https://www.smashingmagazine.com/category/wallpapers/)

- Работает асинхронно.
- Обои на сайте выкладываются каждый месяц, поэтому при запуске нужно указать год, месяц.
- Если месяц не указан, будут скачаны обои за весь указанный год. Год нужно указывать обязательно.
- Можно задать разрешение для скачивания. Если разрешение не указано, будут скачаны обои с разрешением 1920x1080.
- Можно указать папку, куда сохранять обои. Если папка не указана, обои будут сохраняться в папку wallpapers внутри проекта.
- При повторной загрузке одних и тех же обоев, файлы на диске будут перезаписаны. В исходном коде можно раскомментировать возможность для сохранения всех версий файлов.

### Как запустить

1. В корневой папке проекта активировать виртуальную среду:

   `source env/bin/activate`

2. Установить зависимости:

   `pip install -r requirements.txt`

3. Запустить указав ключи:

   `python src/main.py --year 2025 --month 3 --resolution 1920x1080`


   `--year` - год за который надо скачать обои. Нужно указывать обязательно.
   
   `--month` - месяц, за который нужно скачать обои - число от 1 до 12. Если месяц не указан, будут скачаны обои за весь год (за все 12 месяцев).
   
   `--resolution` - разрешение, которое нужно скачать - строка вида "WIDTHxHEIGHT". Если не указано, будут скачаны обои с разрешением 1920x1080.

   `--save-to` - путь к папке, куда сохранять обои. Если папка не указана, обои будут сохраняться в папку wallpapers внутри проекта.
