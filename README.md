## walls-downloader
Это маленькая CLI утилита для скачивания обоев для рабочего стола с сайта [SmashingMagazine](https://www.smashingmagazine.com/category/wallpapers/)

- Работает асинхронно.
- Обои на сайте выкладываются каждый месяц, поэтому при запуске нужно указать год, месяц и желаемое разрешение с использованием ключей --year --month --resolution соответственно.
- Если месяц не указан, будут скачаны обои за весь указанный год. Год нужно указывать обязательно.
- Если разрешение не указано, будут скачаны все разрешения.
- При повторной загрузке одних и тех же обоев, файлы на диске будут перезаписаны. В исходном коде можно расскомментировать возможность для сохранения всех версий файлов.
- По умолчанию обои сохраняются в папку SmashingWallpapers, которая автоматически создается внутри проекта при запуске.

### Как запустить

1. В корневой папке проекта активировать виртуальную среду:

   `source env/bin/activate`

2. Установить зависимости:

   `pip install -r requirements.txt`

3. Запустить указав ключи:

   `python src/main.py --year 2025 --month 3 --resolution "1920x1080"`


   `--year` - год за который надо скачать обои. Нужно указывать обязательно.
   
   `--month` - месяц, за который нужно скачать обои - число от 1 до 12. Если месяц не указан, будут скачаны обои за весь год (за все 12 месяцев).
   
   `--resolution` - разрешение, которое нужно скачать - строка вида "WIDTHxHEIGHT", например "1920x1080". Если не указано, будут скачаны все разрешения.
